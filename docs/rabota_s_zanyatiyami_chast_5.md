# Выдача занятий

**Как выдать занятие?**

- Нажать на кнопку «Выдать занятие ученикам», которая располагается внизу экрана 

*или*


-  Войти в курс
-  Выбрать занятие
-  Нажать на кнопку «Выдать занятие»
