# Добавление задач

**Как добавить задачу? **

- Нажать на кнопку «Добавить задачи»
- Выбрать курс и занятие, из которого необходимо добавить задачи 
- Выбрать нужные задачи, нажав кнопкой мыши на блок задач 
- Нажать на кнопку «Готово»
